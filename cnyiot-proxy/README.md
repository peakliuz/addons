# 晨域电表代理服务

本项目通过调用晨域物联科技[管理后台][cnyiot-boss]接口，实现电量信息查询功能。
以REST Api方式提供晨域智能电表的用电信息，可结合Node-Red或者HomeAssistant命令行调用。
